# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2025-07-08

### Added
- New test cases for emotional language and subjective reasoning
- Enhanced documentation with better examples and usage guides
- Improved GitHub workflow for CI/CD

### Changed
- **ML Suggestions**: Completely revamped suggestion engine for better accuracy
  - More precise emotional language detection
  - Better handling of subjective vs objective statements
  - Improved flow analysis between reasoning steps
  - Reduced false positives in suggestion generation
- **Code Quality**:
  - Refactored test suite for better maintainability
  - Improved type hints and documentation
  - More consistent error handling

### Fixed
- Fixed issue with duplicate suggestions
- Addressed edge cases in flow analysis
- Improved handling of very short reasoning steps

## [0.1.0] - 2025-07-08

### Added
- Initial release of Chain of Thought Debugger
- Core functionality for analyzing reasoning chains
- Support for detecting various logical fallacies
- Comprehensive test suite
- Command-line interface
- ML-based suggestion engine

### Changed
- Improved circular reasoning detection
- Enhanced emotional reasoning analysis
- Better test coverage and reliability

### Fixed
- Various bug fixes and performance improvements
